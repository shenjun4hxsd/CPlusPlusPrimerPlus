##指针小结

1、 声明指针


    double * pn;
    char * pc;

2、 给指针赋值


    double * pn;
    double * pa;
    char * pc;
    double bubble = 3.2;
    pn = &bubble;
    pc = new char;
    pa = new double[30];
    
3、对指针解除引用

    cout << *pn;
    *pc = 'S';
    
    另一种对指针解除引用的方法是使用数组表示法，例如，pn[0]与*pn是一样的。绝不要对未被初始化为适当地址的指针解除引用。
    
4、区分指针和指针所指向的值

如果pt是指向int的指针，则*pt不是指向int的指针，而是完全等同于一个int类型的变量。pt才是指针。

    int* pt = new int;
    *pt = 5;
