##C++11 数组初始化方法

```javascript
    // C++11 可省略=号
    int a[3]{1,2,3};
    
    // 可不在花括号内包含任何东西，所有元素都设置为0
    int b[10]={};
    int c[100]{};
    
    // 列表初始化禁止缩窄转换
    long plifs[] = {25, 92, 3.0}; //❌ not allow  3.0的小数点是0，会缩窄转换
    char slifs[4] {'h', 'i', 1122011, '\0'};  //❌  1122011超出了char的取值范围
    char tlifs[4] {'h', 'i', 112, '\0'};  // 112虽然是数字，但在char的取值范围
```
